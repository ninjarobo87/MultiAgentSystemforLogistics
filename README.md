# Multi-Agent State Machine for Logistics Company using LangGraph

This workspace contains a split implementation of a logistics multi-agent state machine using LangGraph.

## Files

- `architecture_overview.md` - System architecture, processing flow, and ASCII diagram.
- `logistics_state_machine.py` - Complete Python implementation of the LangGraph state machine.
- `state_machine_diagram.md` - Mermaid state diagram of the workflow.
- `architectural_decisions.md` - Key design decisions and rationale.
- `token_bleeding_prevention.md` - Token usage and retry cap strategy.

## Usage

Run `logistics_state_machine.py` to execute the sample scenarios if you have the required LangGraph/LangChain environment configured.
